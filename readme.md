# 量产固件烧录工具 V1.00

> 开发环境:Qt5.9.3(Qt Creator 4.4.1)  MinGW 5.3.0 (32-bit)

通过调用华为UEUpdaterCLI-3.22.0.14和SsbLoader-3.22.0.14分别烧录UE和SSB;

1. 过滤了串口,只选择对应硬件的串口
2. 支持配置华为工具路径
3. 支持中文以及包含空格的外部程序调用
4. 逐条解析外部程序的信息
5. 拥有一个秒表计时器(不精准)
6. 支持四路同时进行
7. 简单的修改外观样式

> 其它: void _slotQueryPortInfo(); // 查询串口信息,需添加QT += serialport
